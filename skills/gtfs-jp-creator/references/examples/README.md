# GTFS-JP サンプルデータ

本スキルの出力「正解形式」を示す GTFS-JP サンプルを置く場所。

## 同梱サンプル

- **`sample_naogata_eimanji/`** — 直方市コミュニティバス 永満寺線（往復・21停留所・9便）。
  本スキルが Excel から生成した実例。標準 Validator ERROR 0・内部整合 100%・P11 座標 100%。
  GTFS-JP 各ファイル（拡張 `agency_jp`/`routes_jp`・`translations`・`shapes` 含む）の書式見本。
  （`agency`/`fare` はPDF外メタのため暫定値・要確認）

## 追加候補

- AIGID提供の公式サンプル／須恵町・柳川市・宗像市 等のコミュニティバス（ライセンス確認後）

## 用途

1. **学習用**: LLMに「こういう形式で出力してほしい」と示すfew-shot exampleとして
2. **評価用**: 生成精度を測るための正解データとして
3. **テスト用**: スクリプトの動作確認用入力データとして

## ステータス

**`sample_naogata_eimanji/` を同梱済み。** 追加サンプルはライセンス確認後に拡充予定。
