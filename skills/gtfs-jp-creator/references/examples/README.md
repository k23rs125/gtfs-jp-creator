# GTFS-JP サンプルデータ

正解データとなるGTFS-JPサンプルを配置する場所。

## 予定するサンプル

- AIGID提供の公式サンプル（県内のバス事業者）
- 須恵町コミュニティバス（先輩研究で扱った題材）
- 柳川市コミュニティバス
- 宗像市コミュニティバス（H30橋本氏の研究で扱われた題材）

## 用途

1. **学習用**: LLMに「こういう形式で出力してほしい」と示すfew-shot exampleとして
2. **評価用**: 生成精度を測るための正解データとして
3. **テスト用**: スクリプトの動作確認用入力データとして
